quora est une plateforme qui permet aux utilisateurs de poser des questions et attendre les réponses des autres. 
Vu le nombre énorme des visiteurs du site Quora, il y a sans doute des questions qui se répètent, alors l'objectif du projet est
l'utilisation du machine learning afin de classifier les questions qui ont le même sens, autrement dit,
trouver les questions similiaires.
